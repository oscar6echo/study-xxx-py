# Study package for course QAHXXX

## Install

```bash
# select env
conda activate study

# dev install
pip install -e .
```

## Run

Pick, edit script in folder [/test](./test):

```bash
# select env
conda activate study

# from repo root dir
cd test
python the-selected-script.py
```

Or run jupyter from folder [/test](./test):

```bash
# select env
conda activate study

# from repo root dir
cd test
jupyter retro
# or
jupyter lab
```
